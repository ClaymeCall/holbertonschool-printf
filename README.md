# _printf

## Description
This project aims to create a function similar to printf. We will name this function _printf

## Compilation/instalation
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 -Wno-format *.c

## Requirements
- '%s' : Prints a string
- '%c' : Prints a single character
- '%%' : Prints '%'
